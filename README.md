**InspectusPro API** — *MVP Backend multi-tenant para laudos com formulários dinâmicos*

Backend de uma plataforma SaaS voltada à geração de laudos técnicos com suporte a formulários dinâmicos, controle de permissões granulares e base estruturada para planos e cupons.

> 🏗️ Arquitetura multi-tenant com isolamento row-level via coluna `tenant_id` e identificação por header `X-Tenant-Id`.

| Camada | Tecnologia |
|---|---|
| Linguagem | Java 21 |
| Framework | Spring Boot 4 · Spring MVC · Spring Security |
| Autenticação | JWT próprio (MVP) |
| Banco de dados | PostgreSQL + Flyway (migrations) |
| ORM | JPA / Hibernate |
| Documentação | OpenAPI / Swagger UI (springdoc) |
| IDs | UUIDv7 (time-ordered — melhor performance em índices) |
| Multitenancy | Row-level isolation via `tenant_id` |
